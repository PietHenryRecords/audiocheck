# Audio RMS Checker für Sprecher 🎙️

Dieses Tool ermöglicht es Sprechern und Autoren, ihre Audiobook-Dateien auf korrekte Lautheit (RMS) zu überprüfen.

- Upload von MP3-Dateien
- Sofortige Analyse
- Anzeige von RMS-Werten und Abweichungen
- Download der Ergebnisse als CSV

Entwickelt von Piet Henry Records.
